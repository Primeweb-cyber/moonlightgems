Project Overview

Moonlight Gems is a concept website created for an academic assignment. The objective is to demonstrate AI-assisted website creation, modern UI/UX principles, and a realistic product catalog experience (search, filtering, product cards, and a quick-view modal).

  • Mobile-first responsive layout
  • Clear navigation and consistent spacing
  • Realistic browsing experience without payments
